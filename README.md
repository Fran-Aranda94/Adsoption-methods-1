# Adsorption of Antibiotics

This project focuses on adsorption using different types of materiales, such polymers or biopolymers

## Objectives
- Study adsorption kinetics
- Evaluate isotherm models (Langmuir, Freundlich, Temkin, and others)

## Pseudo-first-order kinetic modeling
This script fits adsorption kinetic data to the pseudo-first-order model:
qt(t) = qe * (1 - exp(-k1 * t))

Temperatures analyzed:
- 20 °C
- 30 °C
- 40 °C
- 50 °C

Outputs:
- Fitted kinetic parameters (qe, k1)
- R² values
- Experimental vs fitted plots
